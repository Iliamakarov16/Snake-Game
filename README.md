This is simple snake game created with raylib. 
How to play? There are two ways:
1. Using Snake.exe for windows or Snake unix executable for Mac
2. If you want to modify the game you can use VS code. First enter the game directory, then click on "main.code-workspace" file and click "Open Workspace". After this you will be able to run game by opening main.cpp and clicking F5(or fn + F5 if using Mac) to run it.
Controlls:
WASD or arrow keys to move.
R to pause.
